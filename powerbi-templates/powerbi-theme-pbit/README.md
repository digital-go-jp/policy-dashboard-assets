## ライブラリの特徴
1. テキストサイズ、カラー等が整備されたチャート
2. 多種多様なチャート・コンポーネント
3. 整理されたレイアウトをつくるためのグリッドシステム
4. 7色のカラーパレット

詳細は、デジタル庁 「[ダッシュボードデザインの実践ガイドブックとチャート・コンポーネントライブラリ](https://www.digital.go.jp/resources/dashboard-guidebook)」を確認してください。
